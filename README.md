# 🚀 AI Agent MCP-style Server

This project simulates a Model Context Protocol (MCP) server using FastAPI.

## 🔧 Features
- Math tools (add, multiply)
- File operations (read/write)
- Weather API integration

## ▶️ Run Server
uvicorn server:app --reload

## 🧪 Test Client
python client/test_client.py

## 📌 Concept
Implements MCP-style tool calling using REST APIs.

User → AI → Tools → Actions
